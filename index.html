Repos:

  • RPiHTTPServer - Python based web "nano"framework built on top of BaseHTTPServer originally intended to quickly build web interface for Raspberry Pi projects
  • jquery-simpletree - Yet another jQuery treeview plugin
  • jquery-textileToolbar - A jQuery plugin to add a textile toolbar on textareas
  • qrcode-reader - A jQuery plugin implementing a browser interface for the cozmo jsQR QR code reading library
  • BlinkyBuzzer - An Arduino library for managing a blinking led associated with a buzzer/beeper in a non-blocking fashion.
